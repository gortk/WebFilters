This zip file contains a filter list for ublacklist to avoid shitty results when searching for NSFW. 
It avoids trash sites, lure short gallery sites that want to promote major paid sites mostly. 
Other malicious sites are filtered too. The main idea is to remove trash an only show the sites worth your time.
__ALL MAJOR PORN SITES ARE REMOVED TOO__, because _if you wanted to search using a major site, you will go to that site instead
of using a search engine_. This is not whimsical, it is because those sites spam results more than any other making your searches 
impossible. __Some sites that are difficult to scrape using IDM or other methods are removed too.__

### THIS LIST WILL REMOVE TONS OF SITES, SO BE AWARE YOUR SEARCHES WILL BE REALLY SMALL IN RESULTS.
This is not "my fault", its because nsfw sites are mostly trash AND the search engines are trash too. 

## HOW TO INSTALL:
To install, simply unzip and import using ublacklist.
